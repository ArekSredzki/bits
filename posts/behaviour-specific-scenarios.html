Behaviour Specific Scenarios

A definition of target scenarios, in which candidate control/planning systems will be evaluated for tuning specific edge cases.

February 1, 2017 - 2 minute read -
control simulation

In order to effectively design control and planning systems for an agent, we must clearly define the scenarios in which it will be evaluated and proven to be stable. What is described below is a comprehensive test suite and will likely be refined.

Generic obstacle scenario classes

  • No obstacles
  • Generic static obstacle test
    • Static random sparse
    • Static random dense
  • Generic dynamic obstacle test (continuous heading)
    • Other agent bow
    • Other agent stern
    • Other agent port
    • Other agent starboard
    • Multiple other agents, one from port and one from starboard
